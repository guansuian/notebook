# 整个字符串转化

```java
String s = "AbC";

String lower = s.toLowerCase();
String upper = s.toUpperCase();
```

# 单个字符 char 转大小写

```java
char c = 'a';

char upper = Character.toUpperCase(c); // 'A'
char lower = Character.toLowerCase(c); // 'a'
```

# 手动转化

```java
// 小写 → 大写
if (c >= 'a' && c <= 'z') {
    c = (char) (c - 'a' + 'A');
}

// 大写 → 小写
if (c >= 'A' && c <= 'Z') {
    c = (char) (c - 'A' + 'a');
}
```

也可以这样写

```java
c = (char) (c - 32); // 小 → 大
c = (char) (c + 32); // 大 → 小
```

或者是这样写

```java
c ^= 32;
```
如果字母是大写，那么就转化为小写。如果是小写，就转化为大写。
